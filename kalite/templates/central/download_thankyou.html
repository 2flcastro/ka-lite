What do I do after download?

  • When the download completes, copy the file to your server(s).
  • From the server(s), unzip the zip file.
  • From the server(s), run the install_from_zip script file for your operating system

Do you have Python?

Python is the program that runs KA Lite; some systems may not have it.
  • If you need it, download Python (version 2.7).
  • Copy the Python installer over to your server(s).
  • Install Python before you install KA Lite
  • Make sure to add Python to your PATH (Windows, Linux, MacOS)
  • Then, install KA Lite (instructions above)
